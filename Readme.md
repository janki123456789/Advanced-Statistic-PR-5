# 📊 Linear Algebra for Data Analysis  
## Vector, Matrix Operations & Dimensionality Reduction

---

## 📌 Project Overview
This project demonstrates the application of **Linear Algebra concepts in Data Analysis** using Python.  
The objective is to explore how vectors, matrices, and advanced mathematical techniques such as **Eigenvalues, Singular Value Decomposition (SVD), PCA, and LDA** can be used to analyze and transform datasets.

The project is implemented using **Python in Jupyter Notebook (PR-5.ipynb)** and covers both **mathematical theory and practical implementation**.

---

# 🎯 Objective

The main objectives of this project are:

- Understand **vector representation of data**
- Perform **matrix operations**
- Explore **linear transformations**
- Compute **eigenvalues and eigenvectors**
- Apply **dimensionality reduction techniques**
- Visualize vector relationships in **2D and 3D space**

This project helps in building strong foundations in **Linear Algebra for Data Science and Machine Learning**.

---


# 🧮 Part A: Vector & Matrix Fundamentals

### Vector Representation
Each student's subject scores are represented as a **vector**.

Example:

```
Student A = [Math, Science, English]
Student B = [Math, Science, English]
```

### Calculations Performed

✔ Norm-1 and Norm-2 of vectors  
✔ Dot product between vectors  
✔ Angle between students' score vectors  
✔ Cross product (for 3D vectors)  
✔ Projection of one vector onto another  

These operations help measure **similarity and magnitude relationships between vectors**.

---

# 📊 Part B: Matrix Operations

A matrix was created using **students × subjects**.

Matrix operations performed:

- Matrix Addition
- Matrix Multiplication
- Matrix Transpose
- Determinant Calculation
- Matrix Inverse (where possible)

These operations demonstrate how datasets can be manipulated using **linear algebra techniques**.

# 📐 Part C: Linear Transformations & Geometry

### Concepts Explained

- Line in 2D Space
- Plane in 3D Space
- Hyperplane in Higher Dimensions

The project also demonstrates how dimensionality increases:

```
2D → Line
3D → Plane
n-D → Hyperplane
```

This concept is important in **machine learning decision boundaries**.

---

# 📉 Part D: Eigenvalues & Matrix Decomposition

Advanced linear algebra techniques were applied:

### Eigenvalues & Eigenvectors
Computed from the **covariance matrix** to understand variance directions.

### LU Decomposition
Matrix was decomposed into **Lower and Upper triangular matrices**.

### Singular Value Decomposition (SVD)
Used to break the dataset matrix into:

```
U × Σ × Vᵀ
```

SVD plays a major role in **dimensionality reduction and recommendation systems**.

---

# 📉 Part E: Dimensionality Reduction

### Principal Component Analysis (PCA)

PCA was applied to reduce dataset dimensions from **multiple subjects to 2 principal components**.

Benefits:
- Reduces complexity
- Retains maximum variance
- Improves visualization

### Linear Discriminant Analysis (LDA)

Used for classification of students into categories:

- **Above Average**
- **Below Average**

LDA finds the best projection to maximize class separation.
